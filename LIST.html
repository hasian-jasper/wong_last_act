HTML
HTML (HyperText Markup Language) is the most basic building block of the Web. It is the standard markup language for documents designed to be displayed in a web browser. a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
Start Tag, Element Content and End Tag
An opening tag begins a section of page content, and a closing tag ends it. For example, to markup a section of text as a paragraph, you would open the paragraph with an opening paragraph tag [p] and close it with a closing paragraph tag [/p] (closing tags always proceed the element with a /).
[p]Element Content[/p]
DOCTYPE html Declaration
The DOCTYPE declaration informs the web browser of the HTML version used to create the page. The first line of an HTML file contains this declaration. This guarantees that the web page is parsed uniformly across web browsers.
[!DOCTYPE html]
HTML Element
An HTML element is a type of HTML document component that instructs a web browser on how to structure and interpret a specific section of an HTML document. The root element is the html element, which defines the entire HTML document. It has a start tag, an end tag, and a body element within the html element.
[html] [/html]
Head Element
The head element, which is placed between the html and body tags, is a container for metadata (data about data). The head element includes meta data about the document, such as the title, scripts, and style sheets. It primarily stores information for machine processing rather than human reading (the content of the head is not displayed on the page).
[head] [/head]
Title Element
The head tag contains the title element. The title element specifies the title of the document that appears in the title bar of a browser or in the tab of a page. It only has text in it. The title is not part of the document's text, but rather a property of the entire document. It informs website visitors about the content of the page.
[title] [/title]
Body Element
The body element represents the content of an HTML document that will be directly visible on your web page, such as headings, paragraphs, images, hyperlinks, tables, lists, and so on. A document can only have one body element. After the head tag, the body tag must come first. This tag is required for all HTML documents and should only be used once throughout the document.
[body] [/body]
HTML Headings
The header element is a container for introductory content or a collection of navigational links. HTML headings are titles or subtitles that you want to appear on your website. A heading element entails all font changes, paragraph breaks before and after the heading, and any white space required to render the heading. H1 to H6 are the heading elements, with H1 being the highest (or most important) level and H6 being the lowest. The higher the number, the smaller the fonts.
[h1] [/h1], [h2] [/h2], [h3] [/h3], [h4] [/h4], [h5] [/h5], [h6] [/h6]
HTML Paragpraphs
In an HTML document, a paragraph element is made up of content — text, images, or other content such as form fields — that appears between an opening p tag and a closing /p tag. A paragraph always begins on a new line, and browsers add some white space (a margin) before and after it. It is useful because Internet browsers include a small margin before and after each element.
[p] [/p]
HTML Links
A link (short for hyperlink) is an HTML object that allows you to jump to a new location when you click or tap it. The link tag defines the relationship between the current document and an external resource. The link tag is most often used to link to external style sheets or to add a favicon to your website. The link element is an empty element, it contains attributes only.
[a] [/a] or [link] [/link]
HTML Images
The img tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The img tag creates a holding space for the referenced image. The img tag is empty, it contains attributes only, and does not have a closing tag.
[img]
HTML Elements
The HTML element is everything from the start tag to the end tag, The HTML element can contain formatting instructions, semantic meaning, and content. But some HTML elements have no content.  The term "empty element" refers to these components. The end tag is absent from empty elements.
[html] [/html], [head] [/head], [title] [/title], [h1] [/h1] to [h6] [/h6], [p] [/p], [a] [/a]
Nested HTML Elements
Nested tags are tags that are arranged as sub-tags under a main tag. HTML elements can be nested, which means that one element can be inserted inside another element. You can use nesting to apply multiple HTML tags to a single piece of content. The relationship between the DL, DT, and DD tags is a good illustration of nesting.
[p]This is [em] Emphasize [/em] Text [/p]
HTML Attributes
HTML attributes are special words used inside the opening tag to control the behavior of the element. A type of HTML element can be modified by HTML attributes. HTML elements' color, size, and functionality can all be altered using attributes.
href attributes
The HTML href Attribute is used to specify the URL of the page to which the link leads. The HREF contains two components: the URL, which is the actual link, and the clickable text that appears on the page, known as the "anchor text." The href attribute specifies the location (URL) of the external resource (most often a style sheet file).
  • Syntax: [ElementName href="URL" >.....[/ElementName]
  • A: [a href="https://www.w3schools.com"]Visit W3Schools[/a]
  • Base: [base href="https://www.w3schools.com/images/"]
  • Link: [link rel="stylesheet" type="text/css" href="theme.css"]
src attributes
The src attribute specifies the location (URL) of the external resource. There are two ways to specify the URL in the src attribute, the absolute and relative URL.

1. Absolute URL - Links to an external image that is hosted on another website. Example is: "https://www.w3schools.com/images/img_girl.jpg"

2. Relative URL - Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page. Example is: src="/images/img_girl.jpg".

  • Syntax: [ElementName src="URL" ].....[/ElementName]
  • Image: [img src="https://www.w3schools.com/images/lamp.jpg" alt="Lamp" width="32" height="32"]
  • Iframe: [iframe src="/default.asp"][/iframe]
  • Embeded:[embed src="helloworld.swf"]
width & height attributes
  • Syntax: [ElementName width="pixels" height="pixels"]
  • Object: [object data="helloworld.swf" height="400" width="400"][/object]
  • Image: [img src="smiley.gif" alt="Smiley face" height="42" width="42"]
  • Video: [video width="320" height="240" controls][/video]
alt attribute
The alt attribute specifies an alternate text for an specific content like image, if the image cannot be displayed. The alt attribute provides alternative information for an image if a user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
  • Syntax: [ElementName alt="text" ].....[/ElementName]
  • Area: img src="smiley.gif" alt="Smiley face"]
  • Image: [img src="pancakes.png" alt="pancakes"]
  • Input: [input type="image" src="submit.gif" alt="Submit" width="48" height="48"]
style attribute
The style attribute specifies an inline style for an element. The style attribute will override any style set globally, e.g. styles specified in the [style] tag or in an external style sheet.
  • Syntax: [ElementName style="StyleRuless......"]text content[/ElementName]
  • Paragraph: [p style="color:green;font-size:40px;"] Paragraph [/p]
  • Header: [h1 style="color:blue;text-align:center;"]This is a header[/h1]
  • Div: [div style="background: #ffe7e8; border: 2px solid #e66465;"]
HTML Horizontal Break
The [hr] tag is an empty tag that does not need to be followed by an end tag. Used to specify the horizontal rule's alignment. In an HTML page, a horizontal rule or a thematic break is used to divide or separate document sections.
[hr]
HTML Line Break
A line break is a command or sequence of control characters that returns the cursor to the next line and does not create a new paragraph. It is useful for writing a poem or an address, where the division of lines is significant.
[br]
HTML Style Attributes
The style element is a "metadata” type element, which means its purpose is to provide setup for how the rest of the document will be displayed. We use this element to embed style declarations within our HTML document, rather than linking to an external dedicated stylesheet. It is essentially a form of inline styling.
Background Color
The HTML style attribute is used to add styles to an element, such as color, font, size, and more. The style attribute specifies an inline style for an element. The style attribute will override any style set globally, e.g. styles specified in the [style] tag or in an external style sheet. You can aso set background for two diifferent elements.
  • Syntax: [ElementName style="background-color:value;"]text content[/ElementName]
  • Background Color: [body style="background-color:powderblue;"]
  • Header: [h1 style="background-color:powderblue;"]This is a heading[/h1]
  • Paragraph: [p style="background-color:tomato;"]This is a paragraph.[/p]
Font Color
The HTML font color Attribute is used to specify the text color inside the font element. This attribute is used to show enclosed text in a different color. It improves the website's readability and accessibility.
  • Syntax: [ElementName style="font color="value"]text content[/ElementName]
  • Header: [h2 style="color:blue;"]text content[/h2]
  • Paragraph: [p style="color:red;">This is a paragraph.[/p]
Fonts
The HTML font face Attribute is used to specify the font family of the text inside font element. Attribute Values: It contains single value font family which is used to specify the font family.Font to use for text. Listed as one or more font names.
  • Syntax: [ElementName style="font face="font_family"]text content[/ElementName]
  • Header: [h1 style="font-family:verdana;"]text content[/h1]
  • Paragraph: [p style="font-family:courier;"]This is a paragraph.[/p]
Text Size
This attribute is used to adjust the size of the text in the HTML document using a font tag with the size attribute. The range of size of the font in HTML is from 1 to 7 and the default size is 3. The size attribute specifies how large a font will be displayed in either relative or absolute terms. Close the [font] tag with [/font] to return to a normal text size.
  • Syntax: [ElementName style="font size="value"]text content[/ElementName]
  • Header: [h1 style="font-size:300%;"]This is a heading[/h1]
  • Paragraph: [p style="font-size:160%;">This is a paragraph.[/p]
Text Alignment
The text-align attribute is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified. The text-align attribute defines the horizontal text alignment for an HTML element.
  • Syntax: [ElementName style="text-align:value;"]Text Content[/ElementName]
  • Header: [h1 style="text-align:center;"]Centered Heading[/h1]
  • Paragraph: [p style=text-align:right;"]Right-aligned paragraph.[/p]
HTML Formatting Elements
Formatting elements were created to display different types of text. HTML text formatting tags can be used to bold, italicize, underline, and do other things. Various HTML Formatting tags can alter the appearance of text on a web page and make it more readable.
Bold Text
The HTML [b] tag specifies bold text with no logical significance. It begins with b and ends with /b tag. .
  • Syntax: Bold: [ElementName [b]Text Content[/b][/ElementName]
  • Paragraph: Bold: [p]This is normal text [b]and this is bold text[/b][/p]
Important Text
The HTML strong tag is used to define text with strong importance. The content inside is typically displayed in bold.
  • Syntax: Strong: [ElementName [strong] Text Content[/strong][/ElementName]
  • Paragraph: Strong: [p]This is some normal paragraph text, [strong] and this is some important text.[/strong][/p]
